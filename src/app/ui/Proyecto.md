# Knowledge Graph React

## Propósito
Landing page minimalista para visualizar grafos de conocimiento con Neo4j Design Library.

## Comandos de Ejecución

```bash
cd ui
npm install
npm start
```

## Estructura Simplificada

- ✅ React 18
- ✅ Neo4j NVL (visualización de grafos)
- ✅ Datos mockeados
- ✅ Interacción con clicks

## Archivos Esenciales

```
ui/
├── src/
│   ├── components/
│   │   └── GraphVisualization.js  (componente principal del grafo)
│   ├── data/
│   │   └── mockData.js            (datos del grafo)
│   ├── App.js                      (aplicación)
│   └── index.js                    (punto de entrada)
└── package.json
```

La aplicación se abrirá en http://localhost:3000
